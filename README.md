# 🔬 AI Research Assistant

> An intelligent research companion built with LangChain, Groq, FAISS, and Streamlit.
> Ask questions, summarize documents, analyze text — all locally and for free.

## ✨ Features

- 💬 Conversational Chat — ask anything with persistent memory per session
- 📄 RAG (Retrieval Augmented Generation) — upload PDFs and chat with them
- 📝 Text Summarizer — quick and structured summaries of any text
- 🔍 Text Analyzer — word count, reading time, study time, keywords
- 🎓 Beginner / Expert Mode — adjusts explanation depth on the fly
- 🧠 Session Memory — remembers your conversation history
- 🔒 100% Local Embeddings — your documents never leave your machine
